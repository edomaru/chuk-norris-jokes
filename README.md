# Chuk Norris Jokes

Create chuk norris jokes in your next PHP project

## Installation

Install the package using this command:

```bash
composer require masaruedo/chuk-norris-jokes
```

## Usage

```php
use Masaruedo\Jokes\JokeFactory;

$joke = new JokeFactory();

$joke->getRandomJoke();
```
